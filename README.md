# TP15 – Expressions Lambda en Java (JDK 17+)

Ce projet contient une série d'exercices pour comprendre :
- les expressions lambda
- les interfaces fonctionnelles (java.util.function)
- les références de méthodes
- l'API Stream (basique + avancée)
- la composition de fonctions
- la gestion d'exceptions en contexte fonctionnel
- un validateur basé sur des lambdas
- TriFunction, comparateurs composés, factorielle en IntStream, memoization
